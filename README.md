# 💡 API LAGE IMÓVEIS

API desenvolvida para funcionamento em conjunto ao site para a imobiliária Lage Imóveis. 
Disponível [neste link](https://lageimoveis.com.br)

## 📲 Tecnologias utilizadas 🖥

### 🧠 NodeJs
- Framework
### 🍃 MongoDB
- Banco de dados
### 💾 AWS S3
- Armazenamento das imagens
